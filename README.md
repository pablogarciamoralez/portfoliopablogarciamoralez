# Hola Mundo

Soy Pablo García Moralez, Desarrollador Web Full Stack y este es mi portfolio personal.
Fue realizado con React y Bootstrap y cuenta con un formulario controlado con javascript para envío de mails mediante email.js.
Para ejecutar este proyecto en tu PC:

1. Descargá el proyecto
2. Abrí el directorio donde lo guardaste
3. Una vez estes en el directorio de tu proyecto, con tu app de consola preferida ejecutá el comando **npm i** para instalar las dependencias necesarias.
4. Ejecutá el comando **npm start** y el proyecto se abrirá en tu navegador. También podes visitar el deploy https://pablogarciamoralezdeveloper.vercel.app/
